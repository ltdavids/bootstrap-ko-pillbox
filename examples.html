bootstrap-ko-pillbox

A multi-select knockout component that plays nice with Bootstrap.

Basic

Pillbox requires 2 parameters optionValues and optionText.

If the selectedOptions parameter is set to a valid knockout observableArray, Pillbox will bind to it for selected values.

If the selectedOptions parameter is ommited, Pillbox will create an observableArray internally and expose it via its selectedOptions property.

HTML Markup

Using internal selected options observable
<pillbox params="optionValues:Movies,  optionsText:'text'"/>
Using an observableArray in the view model
<pillbox
    params="optionValues:Movies, selectedOptions:SelectedMovies, optionsText:'text'">
</pillbox>

Access internal selected options array with JavaScript

var selected = $('#pb4').data('pillbox').selectedOptions;

Control Sizing

Layout and size can be controlled by applying the standard bootstrap classes .form-inline, .form-horizontal, .form-group and .form-control

HTML Markup

<div class="form-inline">
    <div class="form-group-lg">
        <pillbox id="pb12" class="form-control"
            params="optionValues:Movies, selectedOptions:SelectedMovies, optionsText:'text'">
        </pillbox>
    </div>
</div>

Placeholder

Any text inside the pillbox element will be used as the placeholder. The placeholder can also be set using javascript.

If no placeholder is set then it will default to "Choose"

Choose Movies

HTML Markup

<pillbox id="pb4"
    params="optionValues:Movies,
        selectedOptions:SelectedMovies,
        optionsText:'text'">Choose Movies</pillbox>

JavaScript

$('#pb4').data('pillbox').option('placeHolder','Choose Movies');

Type Ahead

If the typeAhead property is set, Pillbox provides basic typeahead and autocomplete using array filtering.

HTML Markup

<pillbox id="pb2"
         params="optionValues:Movies,
                 selectedOptions:SelectedMovies,
                 optionsText:'text',                             
                 placeHolder:'Classes',
                    typeAhead:true"/>

JavaScript * Future Release

$('#pb2').data('pillbox').option('typeAhead',true);

Show Selected

By default, selected items are filtered from the dropdown. Pillbox can be configured to show selected items in the dropdown by setting the showSelected option.

When set, items that have been selected are shown highlighted. Clicking a selected item will un-select it.

HTML Markup

<pillbox id="pb5" 
        params="optionValues:Movies,
            selectedOptions:SelectedMovies,
            optionsText:'text', 
            placeHolder:'Classes',                             
            typeAhead:true,
                    showSelected:true" />

JavaScript

$('#pb5').data('pillbox').option('showSelected',true);

Templates

Pillbox supports named templates in the same fashion as knockout template binding.

If the value passed in the optionTemplate parameter is a string value then Pillbox will expect a valid html template in the DOM with an id matching the parameter value. If the parameter is a funtion in the viewmodel, Pillbox will call that funtion to obtain the template markup.

The template markup is inserted into the DOM prior to binding being applied by knockout.

HTML Markup

<pillbox id="pb3"
        params="optionValues:Movies,
            selectedOptions:SelectedMovies,
            optionsText:'text',                             
            placeHolder:'Classes',
            typeAhead:true,                             
            showselected:true,
                    optionTemplate:'class-template'" />

Template Markup

<script type="text/html" id="class-template">
    <div class="row" style="width:285px;">
        <div class="col-xs-4"> <img data-bind="attr:{src:img}" style="width:100%;" /></div>
        <div class="col-xs-8" style="padding-left:0">
            <div data-bind=" text:text" style="font-weight:bold"></div>
            <div class="cast" style="color:#000;font-size:11px; line-height:11px" 
                    data-bind="text:cast"></div>
            <div class="item-description" 
                    style="color:#808080;font-size:10px; line-height:9px; margin-top:8px;" 
                    data-bind="text:description">
            </div>
        </div>
    </div>
</script>

Dropdown Height

By default, Pillbox does not constrain the height of the drop down. When using templates or when there are a large number of options it may be desirable to limit the number of rows displayed.

The dropdownRows option can be used to configure the number of items displayed in the dropdown.

Note: Pillbox currently calculates the overall height based on the height of the first item in the list. Results could vary if items are not of equal height.

HTML Markup


<pillbox id="pb8" 
        params="optionValues:Movies,
        selectedOptions:SelectedMovies,
        optionsText:'text',
        placeHolder:'Choose Movies',
        typeAhead:true,
        showselected:true,
        optionTemplate:'class-template',
                    dropdownRows:3" />

JavaScript 

$('#pb2').data('pillbox').option('dropdownRows',3);

Popovers

<pillbox id="pb9" class="form-control"
        params="optionValues:Movies,
        selectedOptions:SelectedMovies,

        placeHolder:'Choose Movies',

$('#pb2').data('pillbox').option('popover',{template:'popover-template'});

Hide Pillbox

There may be situations where you want full control over the display of selected items using an observableArray in your viewmodel.

You can supress the display of the pillbox by using the showPillbox parameter and manage the selected items display on your own.

HTML Markup

<pillbox id="pb10" 
         params="optionValues:Movies,
                 selectedOptions:SelectedMovies,
                 optionsText:'text',
                 placeHolder:'Choose Movies',
                 optionTemplate:'class-template',
                 showSelected:true,
                 dropdownRows:3,
                    showPillbox:false" />

JavaScript

$('#pb7').data('pillbox').option('showPillbox',false);
